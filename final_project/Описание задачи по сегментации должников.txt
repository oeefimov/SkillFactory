Основной целью проекта (задача №1) является сегментация должников энерго-сбытовой компании 
в целях проведения с ними разных мероприятий.

Задачами проекта являются:
1) Реализация алгоритма сегментации и проверка его на тестовых данных
для последующего применения на полном объеме должников.
2) Выдача рекомендаций по проведению мероприятий с сегментами должников,
полученных на тестовых данных.

Данные для проекта были выгружены из рабочей базы ЭСК. Они содержат:
1) Информацию о наличии контактов для оповещении должников;
2) Сумму начислений за электроэнергию;
3) Суммы оплаты за 3, 6, 9, 12, 15, 18 последних месяцев;
4) Количество судебных дел по взысканию долга;
5) Количество месяцев задолженности;
6) Последние даты проверки электрических счетчиков.

Этапами работы над проектом стали:
1) Чтение, анализ, преобразование и очистка данных;
2) Визуализация распределения признаков;
3) Кластеризация классическим DL-методом K-Means и визуализация результатов;
4) Понижение размерности методом построения сети (карты) Кохонена;
5) Кластеризация K-Means карты Кохонена и визуализация результатов;
6) Понижение размерности применением нейронной сети-автокодировщика;
7) Кластеризация скрытого слоя различными DL-методами (K-Means, BIRCH,
Gaussian Mixture, Agglomerative Clusterization) и визуализация результатов;
8) Построение ансамбля по голосованию результата разных моделей кластеризации;
9) Вычисление статистик отдельных кластеров и применение на их основе 
шаблона работы с должниками для выдачи рекомендаций.
