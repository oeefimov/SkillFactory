Основной целью проекта (задача №2) является построение прогноза оплаты долга
со стороны должников энерго-сбытовой компании в целях проектирования бюджета.

Задачами проекта являются:
1) Реализация алгоритма прогнозирования и проверка его на тестовых данных
для последующего применения на полном объеме данных
2) Выдача прогноза поступления оплат от должников на тестовых данных.

Данные для проекта были выгружены из рабочей базы ЭСК. 
Они содержат временной ряд поступления оплат с детализацией по дням. 

Этапами работы над проектом стали:
1) Чтение, анализ, преобразование данных;
2) Визуализация временного ряда;
3) Выявление сезонных составлящих: тренда, сезонности, остатка;
4) Анализ стационарности ряда для определения метода прогнозирования;
5) Построение прогноза DL-методом SARIMAX для цельного ряда;
6) Построение прогноза DL-методом SARIMAX для отдельных составляющих ряда и 
последущего их суммирования;
7) Построение прогноза с применением сверточных нейронных сетей;
8) Построение прогноза с применением рекуррентных нейронных сетей;
9) Построение прогноза с применением комбинированных (свертка + рекурсия) нейронных сетей;
10) Построение прогноза с применением модели Prophet от Facebook 
с учетом выходных дней и сезонности разного уровня;
11) Построение прогноза с применением ансамбля из всех моделей; 
12) Построение прогноза и выдча результата для будущего интервала времени.
